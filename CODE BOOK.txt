==========
Version 0.1 / 30JAN2020 for coursera.org 

Two main tidy dataset are provided:

1) dt_total
------------
It constains the original raw data, reorganized in 1716 variables (columns),
in a tidy dataset.
10299 observations are available.

==> See FEATURES.txt for details

2) dt_stats
------------
It contains, in a tidy data set, for each activity and each subject,
the average of all mean and standard deviation variables

==> See FEATURES_stats.txt for details
